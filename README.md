# ☕ Coffee Shop Sales Dashboard

An interactive Excel dashboard analyzing coffee shop sales performance — built using pivot tables, slicers, and dynamic KPI cards to surface trends across products, stores, and time.

![Dashboard Screenshot](Coffee-sales-Dashboard.png)

## Key Metrics
- **Total Sales:** ₹6,98,812
- **Total Transactions:** 1,49,116
- **Total Quantity Sold:** 2,14,470

## Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers (Day-wise, Month-wise filtering)
- GETPIVOTDATA for dynamic KPI cards
- Conditional formatting & custom chart styling

## Features
- Interactive filtering by day of week and month
- Real-time KPI cards that update with slicer selection
- Multi-chart view: monthly trends, product-type sales, store-wise comparison, top 10 selling products, hourly sales, and transaction patterns
- Clean, presentation-ready layout

## How to Use
Download the `.xlsx` file and open it in Excel. Click on the Day or Month slicer buttons to filter the entire dashboard dynamically.
